# PR Template

Copy this to `.github/PULL_REQUEST_TEMPLATE.md` in each repo.

```markdown
## Epic

<!-- Required: Link to Notion Epic -->
[Epic: Title](notion-url)

## Project

- [ ] Backend
- [ ] Mobile
- [ ] Infrastructure
- [ ] AI
- [ ] Frontend

## Summary

<!-- Be concise. What, why, and key changes. -->

## Checklist

- [ ] Self-reviewed
- [ ] Build passes
- [ ] No secrets committed
- [ ] Tests updated (if needed)
- [ ] Docs updated (if needed)
```

## PR Checklist (Quick Reference)

Before submitting:

- [ ] Branch up to date with `main`
- [ ] Build passes locally
- [ ] Links to Notion Epic in PR description
- [ ] No secrets/env files committed
- [ ] Reviewed and approved
